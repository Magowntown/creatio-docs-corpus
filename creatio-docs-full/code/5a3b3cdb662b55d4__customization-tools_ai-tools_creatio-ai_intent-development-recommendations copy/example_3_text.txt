Execute the function [Check email settings] that has the following parameters: <ContactID>: a unique ID of the Contact to whom the email can be sent (usually this ID is called Contact and is located on the page where the user is working).<AccountID>: a unique ID of the account to whom the email can be sent (usually this ID is called Account and is located on the page where the user is working). One of these parameters can be empty. In response, you will receive a signal indicating whether you can proceed and with which ID.
