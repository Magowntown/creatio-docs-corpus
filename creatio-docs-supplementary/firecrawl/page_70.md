On this page

Level: beginner

Out of the box, a new section page in the mobile application contains a single **General information** group of columns. This group of columns cannot be renamed or deleted. Existing section pages have additional columns present.

You can add the rest of columns, column groups, and details to the section page manually.

## Add a column to page [​](https://academy.creatio.com/docs/8.x/mobile/mobile-app-setup/set-up-mobile-application-section-page\#title-238-1 "Direct link to Add a column to page")

1. Click ![btn_system_designer.png](https://d3a7ykdi65m4cy.cloudfront.net/ac-en/s3fs-public/images/NoCode_Customization/mobile_page_setup/btn_system_designer.png) to **open the System Designer**.

2. Go to the **System setup** block → **Mobile application wizard**.

3. **Select the workplace** to edit → **Open**.

4. **Click** **Set up sections**.

5. **Select a section** in the list → **Page setup**.

6. **Click** **New column** in the column group to add a field (Fig. 1).
Fig. 1 Add fields to a page

![Fig. 1 Add fields to a page](https://d3a7ykdi65m4cy.cloudfront.net/ac-en/s3fs-public/images/NoCode_Customization/mobile_page_setup/scr_new_column.png)

7. **Select a column** in the **Select column** window → **Select** (Fig. 2).
Fig. 2 Select a column to add to the page

![Fig. 2 Select a column to add to the page](https://d3a7ykdi65m4cy.cloudfront.net/ac-en/s3fs-public/images/NoCode_Customization/mobile_page_setup/scr_select_a_column_0.png)


To change the order of the fields on the page, drag them using the left mouse button.

8. **Save the changes** (Fig. 3).
Fig. 3 Save a mobile app section page that contains the new column

![Fig. 3 Save a mobile app section page that contains the new column](https://d3a7ykdi65m4cy.cloudfront.net/ac-en/s3fs-public/images/NoCode_Customization/mobile_page_setup/scr_save_the_column.png)

9. **Click** **Save** on the Mobile Application Wizard page.


As a result, the changes will become available for all users.

## Add a column group to page [​](https://academy.creatio.com/docs/8.x/mobile/mobile-app-setup/set-up-mobile-application-section-page\#title-238-2 "Direct link to Add a column group to page")

You can add column groups to sections, including groups that are not available in the primary application. The **General information** group cannot be renamed or deleted, but you can add more columns to it.

To add a column group to the section page:

1. **Perform steps 1–5** of the [Add columns to page](https://academy.creatio.com/documents?id=1394#title-238-1) block.

2. **Click** **New** → **Column group** (Fig. 4).
Fig. 4 Add a column group

![Fig. 4 Add a column group](https://d3a7ykdi65m4cy.cloudfront.net/ac-en/s3fs-public/images/NoCode_Customization/mobile_page_setup/scr_new_column_group.png)

3. **Enter a title and a name** for the column group.


|     |     |
| --- | --- |
| **Field** | **Field value** |
| Title | Specify the name to display in the mobile app. |
| Name | Specify the internal name of the column group. You cacn use only Latin letters without special characters. |

4. **Click** **OK**. A new empty column group will appear on the section page (Fig. 5).
Fig. 5 Add columns to a column group

![Fig. 5 Add columns to a column group](https://d3a7ykdi65m4cy.cloudfront.net/ac-en/s3fs-public/images/NoCode_Customization/mobile_page_setup/scr_place_of_work.png)

5. Add columns to the column group.

6. **Save the changes**.

7. **Click** **Save** on the Mobile Application Wizard page.
