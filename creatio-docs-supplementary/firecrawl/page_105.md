On this page

Level: beginner

Records in a mobile app list consist of the following data blocks: title, subtitle, and additional columns (Fig. 1).

The title can display only one column out of the box. All the other blocks can display any data as per your settings.

You can display a maximum of 13 lookup columns in a list. Columns of other types have no restrictions.

Fig. 1 Columns of the Contacts section list in the mobile application

![Fig. 1 Columns of the Contacts section list in the mobile application](https://d3a7ykdi65m4cy.cloudfront.net/ac-en/s3fs-public/images/NoCode_Customization/mobile_list_setup/scr_record_components_0.png)

## Set up the list [​](https://academy.creatio.com/docs/8.x/mobile/mobile-app-setup/set-up-mobile-application-section-list\#title-239-1 "Direct link to Set up the list")

1. Click ![btn_system_designer.png](https://d3a7ykdi65m4cy.cloudfront.net/ac-en/s3fs-public/images/NoCode_Customization/mobile_list_setup/btn_system_designer.png) to **open the System Designer**.

2. Go to the **System setup** block → **Mobile application wizard**.

3. **Select the needed section** → **List setup** (Fig. 2).
Fig. 2 Set up a section list

![Fig. 2 Set up a section list](https://d3a7ykdi65m4cy.cloudfront.net/ac-en/s3fs-public/images/NoCode_Customization/mobile_list_setup/scr_mobile_application_wizard.png)

4. **Click** **New column** and select a column from the list. You can drag columns to change their positions.
Fig. 3 Add columns to a section list

![Fig. 3 Add columns to a section list](https://d3a7ykdi65m4cy.cloudfront.net/ac-en/s3fs-public/images/NoCode_Customization/mobile_list_setup/scr_section_list.png)

5. **Save the changes**.


As a result, Creatio will display new data in the section list (Fig. 4).

Fig. 4 Customized list of contacts in the Creatio mobile app

![Fig. 4 Customized list of contacts in the Creatio mobile app](https://d3a7ykdi65m4cy.cloudfront.net/ac-en/s3fs-public/images/NoCode_Customization/mobile_list_setup/scr_customized_list.png)
