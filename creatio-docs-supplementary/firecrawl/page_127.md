Release date: 03/14/2025

At Creatio, we are committed to empowering our customers with industry-leading product innovations for workflow automation, no-code development, and modern CRM. Today we are advancing new forms of innovation with Creatio version 8.2.2 "Energy," featuring the **following new capabilities and enhancements**.

A comprehensive overview of technical changes and enhancements can be found in the [8.2.2 Energy Changelog](https://academy.creatio.com/docs/8.x/resources/releases/changelog/8-2-2-energy-changelog).

The update guide for the on-site applications is available in a separate [article](https://academy.creatio.com/documents?id=2495).

## Creatio composable apps [​](https://academy.creatio.com/docs/8.x/resources/release-notes/8-2-2-energy-release-notes\#title-2782-1 "Direct link to Creatio composable apps")

### Email Marketing [​](https://academy.creatio.com/docs/8.x/resources/release-notes/8-2-2-energy-release-notes\#title-2782-119 "Direct link to Email Marketing")

**Schedule emails**. You can now ensure the messages reach their audience at the optimal moment by scheduling emails to be sent at a specific time.

**Copy bulk emails**. You can now reuse existing setups without having to configure them from scratch by copying bulk emails.

**Save email rows**. You can now maintain design consistency and save time on repetitive tasks by saving specific rows in the Email Designer.

### Lead Generation [​](https://academy.creatio.com/docs/8.x/resources/release-notes/8-2-2-energy-release-notes\#title-2782-122 "Direct link to Lead Generation")

**Landing Pages in Creatio**. You can take the most out of Creatio landing pages by making your own visually engaging landing pages quickly to capture and convert prospects. Add web forms to streamline lead generation, incorporate images and videos, and structure your content using flexible layout. Once your page is ready, you can download its HTML code for seamless publishing.

### Knowledge Management [​](https://academy.creatio.com/docs/8.x/resources/release-notes/8-2-2-energy-release-notes\#title-2782-120 "Direct link to Knowledge Management")

**Version history**. You can now create article drafts and publish new versions when they are ready to be available for all users. It is also possible to review previous versions of articles and set up a review process for drafts based on your company regulations.

**Article formatting improvements**.

- Table of contents. Navigate long articles effortlessly using an auto-generated outline.
- Formatted quotations. Highlight important references and external sources.
- Code snippets. Display code examples with proper formatting for better readability.

You can add these options to any other rich text field using low-code tools. Learn more: [Add additional formatting options to the toolbar of the "Rich text" type field](https://academy.creatio.com/documents?id=15379&anchor=title-15379-7).

### Customer 360 [​](https://academy.creatio.com/docs/8.x/resources/release-notes/8-2-2-energy-release-notes\#title-2782-121 "Direct link to Customer 360")

**Preferred language filtering**. The **Preferred languages** field on the contact page now displays only the languages specified as used in the **Customer languages** lookup.

### Sales [​](https://academy.creatio.com/docs/8.x/resources/release-notes/8-2-2-energy-release-notes\#title-2782-140 "Direct link to Sales")

**New periods**. Forecast periods are now available up to 2035.

**Link validation**. Creatio AI now validates the provided Creatio links to ensure a smooth experience.

### WCAG compliance [​](https://academy.creatio.com/docs/8.x/resources/release-notes/8-2-2-energy-release-notes\#title-2782-15 "Direct link to WCAG compliance")

**Keyboard controls for components**.

- Keyboard navigation in the **Calendar**, **Message composer**, and **Feed** was improved.

- Skip the **Playbook** and **Timeline** by pressing Tab or press Enter to navigate through them.

- Navigate through multi-select lookup, tags, lookup quick filter, folder tree, breadcrumbs in widgets, rich text formatting options, and field selection windows using the keyboard.
Navigating through field selection window

![Navigating through field selection window](https://d3a7ykdi65m4cy.cloudfront.net/ac-en/s3fs-public/images/Release_notes/release_notes_8_2_2/gif_field_selection_window.gif)

- Navigate not only through list cells but also all list elements, including header cells and selection checkboxes, via keyboard.

- Links, checkboxes, and menus now support keyboard focus.

- Components selected by keyboard navigation now have focus frames.


**Widgets**.

- Values of the Bar/Column widgets now have borders for better contrast and readability.

- Unselected widget elements now have opacity effects during keyboard navigation for better readability.

- Selected widget elements now have better contrast during keyboard navigation for better readability.

- Widgets now have border radius applied for better readability.

- Widgets now have borders and white layer for better contrast and readability.

- Text value of stacked bars no longer changes the color on mouse hover.

- View aggregated chart values as data table.

- Navigate the Gauge and Metric widgets using the keyboard.

- Line, spline, area, and scatter type widgets now display specific values as different shapes instead of circles.

- Editable list panel and bulk action panel are now WCAG compliant. When you move focus through the page the panel itself is focused first. Press Enter to start interacting with its elements or just move focus further through the page. When you are interacting with the panel elements press Esc to move focus out to the level of panel itself.

**Browser scroll bar**. You can now use the browser scroll bar inside the Creatio instance. Set it up on your profile page. System administrators can turn it on for everyone in the "Use default browser scrollbar styles" ("UseDefaultBrowserScrollbarStyles" code) system setting.

## No-code tools [​](https://academy.creatio.com/docs/8.x/resources/release-notes/8-2-2-energy-release-notes\#title-2782-9 "Direct link to No-code tools")

### Application hub [​](https://academy.creatio.com/docs/8.x/resources/release-notes/8-2-2-energy-release-notes\#title-2782-150 "Direct link to Application hub")

**Work with inherited elements**. You can now store extensions of different app elements in a custom app using dependencies on inherited pages, sections, data models, and business processes.

### Business rules [​](https://academy.creatio.com/docs/8.x/resources/release-notes/8-2-2-energy-release-notes\#title-2782-3 "Direct link to Business rules")

**Formulas**. You can now use formulas in the “Set field value” business rule.

**User roles in business rules**. You can now set conditions for all existing business rule actions based on the roles of the current user.

**Streamlined business rule setup**. "Dynamic filter" and "Set value" business rules no longer need conditions to work. This lets you create more advanced business rules easier. Creatio generates the required logic automatically. For existing rules, change them to apply the new behavior.

**Business rule activation without page reload**. You no longer need to reload a page explicitly to activate new business rules. Simply closing and reopening the page is enough.

**Different triggers**. Business rules are now triggered differently depending on whether an attribute in condition was added from the "Attributes" menu item or from the "Data sources" menu. This provides more flexibility in advanced cases.

### Freedom UI Designer [​](https://academy.creatio.com/docs/8.x/resources/release-notes/8-2-2-energy-release-notes\#title-2782-10 "Direct link to Freedom UI Designer")

**Copy page**. You can now copy an existing page to create a new page that has the same layout and components.

**Search for components**. You can now search for a component in the component panel.

**Multi-select lookup improvements**.

- Enable users to add new values in the component.
- Enable users to select values in the component from a selection window.
- Select from multiple columns to display if the multi-select value storage has more than one column related to the object.

**Next best offer layout**. **Next best offers** component now works well with all page layouts.

**Link Next best offer**. You can now link the **Next best offer** component to a specific Freedom UI page, for example, product page.

**Save data without closing the page**. You can now let users stay on the page after they save a new record. The functionality is available for full-sized form pages.

**Save successful message**. You can now explicitly specify that a record is saved by turning on the bar that contains the success message. Assistive technologies receive this information as well.

**Calendar synchronization account**. You can now set up synchronization options for Google or Outlook calendar using the new **Set up calendar synchronization account** button action.

**Permanent toolbar in the Rich text field**. It is now possible to make the toolbar of the **Rich text** field be displayed permanently while editing. This makes it easier to work with large amounts of content.

**Performance**. Freedom UI Designer now opens and saves pages up to 2.5 times faster.

### Integrations [​](https://academy.creatio.com/docs/8.x/resources/release-notes/8-2-2-energy-release-notes\#title-2792-130 "Direct link to Integrations")

**Automatic SSO configuration for new providers**. You can now configure Google SSO and Ping Identity by importing their metadata files into Creatio.

**Spaces in JSON**. Creatio is now able to parse JSON answers from REST web services. This lets you integrate with a wider range of REST web services.

### Mobile Creatio [​](https://academy.creatio.com/docs/8.x/resources/release-notes/8-2-2-energy-release-notes\#title-2782-16 "Direct link to Mobile Creatio")

**Creatio AI**. You can now use Creatio AI in the mobile app.

**Decimal and thousand separators**. Handling of numeric values based on local settings for decimal and thousand separators was improved.

**Predefined filters**. You can now use predefined filters on Freedom UI pages for faster and more consistent filtering.

## Administration​ [​](https://academy.creatio.com/docs/8.x/resources/release-notes/8-2-2-energy-release-notes\#title-2782-14 "Direct link to Administration​")

**Translation improvements**.

- Update translations in the background. To do that, run the "Actualize translations" business process in the process library. As with any other business process, you can track its status in the process log.
- Apply translations in the background. To do that, run the "Apply Translations" business process in the process library. As with any other business process, you can track its status in the process log.
- Apply translations only for a specific language. This is useful if you want to apply translations faster or do not want to apply some languages because translations are not verified. To do this, run "Apply Translations" business process, select the Apply translations for specified language checkbox, and specify the language in the lookup field.
- Force apply translations even if the value of the isChange table column is "false." This is useful if you want to make data between runtime and translation sections consistent. To do this, run "Apply Translations" business process, specify the language, and select the Force apply all translations checkbox.
- Translation application and update actions are now up to 5 and 13 times faster, respectively.

**Page permission security**. To provide an appropriate level of security, the "Can use bypass page opening restrictions" ("CanBypassPageOpeningRestrictions" code) system operation is disabled by default for external users.

**Email Listener security**. To improve stability and security, the microservice now uses a new Linux base image and .NET 8.

## Beta testing of new features​ [​](https://academy.creatio.com/docs/8.x/resources/release-notes/8-2-2-energy-release-notes\#title-2782-17 "Direct link to Beta testing of new features​")

Important

The features below are available for beta testing in Creatio version 8.2.2 Energy. If you have any feedback, contact us at: `beta@creatio.com`. All feedback is appreciated.

**Publish landing pages**. You can now publish landing pages created in the Landing Page Designer directly to a live pre-configured URL. To activate the functionality, turn on the "LandingPagePublishingEnabled" feature. Learn more: [Manage an existing additional feature](https://academy.creatio.com/documents?id=15631).

**Item wrapping**. You can now enable the **Multi-select lookup** component to wrap selected values to the next line instead of hiding them behind a counter. To activate the functionality, turn on the "EnableWrapInMultiSelect" feature. Learn more: [Manage an existing additional feature](https://academy.creatio.com/documents?id=15631).

**Access management for Creatio AI Skills**. You can now leverage security and control costs of executed AI Skills by assigning or revoking execution permissions for specific users or groups. This functionality is available for AI Skills both in "API" and "Chat" usage modes. To activate the functionality, turn on the "GenAIFeatures.UseSkillSchemaOperationRights" feature. Learn more: [Manage an existing additional feature](https://academy.creatio.com/documents?id=15631).
