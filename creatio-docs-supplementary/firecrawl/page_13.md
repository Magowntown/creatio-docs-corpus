On this page

Release date: 07/27/2021

At Creatio, we are working constantly to deliver advanced low-code/no-code capabilities to accelerate your process automation and application development for customer-facing, IT, and back-office teams. Here are the **new features** included in Creatio version 7.18.2.

The update guide for the on-site applications is available in a separate [article](https://academy.creatio.com/node/143/).

## Low-code platform [​](https://academy.creatio.com/docs/8.x/resources/release-notes/7182-release-notes\#title-2070-1 "Direct link to Low-code platform")

### Business processes [​](https://academy.creatio.com/docs/8.x/resources/release-notes/7182-release-notes\#title-2070-2 "Direct link to Business processes")

#### Process activity creation [​](https://academy.creatio.com/docs/8.x/resources/release-notes/7182-release-notes\#title-2070-3 "Direct link to Process activity creation")

- It is now possible to turn on and off activity creation in interactive business process elements. If you do not need to create an activity, specify this in the element setup area.

- An option to connect process activities to other entities was added, streamlining the activity workflow. This feature is available in the **User actions** group's process elements, including the automated emails sent via the **Send email** element.
Activity setup in the Send email element

![Activity setup in the Send email element](https://academy.creatio.com/docs/sites/en/files/images/Release_notes/release_notes_7_18_2/scr_create_activity_in_a_business_process.png)


### User customization tools [​](https://academy.creatio.com/docs/8.x/resources/release-notes/7182-release-notes\#title-2070-4 "Direct link to User customization tools")

#### Homepage setup [​](https://academy.creatio.com/docs/8.x/resources/release-notes/7182-release-notes\#title-2070-5 "Direct link to Homepage setup")

- The existing homepage charts can now be copied. To do this, use the ![](https://academy.creatio.com/docs/sites/en/files/images/Release_notes/release_notes_7_18_2/btn_homepage_copy_element.png) button or Ctrl+C and Ctrl+V hotkeys.
- Moving, resizing, and deleting the homepage elements will push up the dashboards below if there is room on the canvas.

Setting up the homepage

![Setting up the homepage](https://academy.creatio.com/docs/sites/en/files/images/Release_notes/release_notes_7_18_2/gif_edit_homepage.gif)

### Base interface and system capabilities [​](https://academy.creatio.com/docs/8.x/resources/release-notes/7182-release-notes\#title-2070-6 "Direct link to Base interface and system capabilities")

#### Chats [​](https://academy.creatio.com/docs/8.x/resources/release-notes/7182-release-notes\#title-2070-7 "Direct link to Chats")

- Chats can now be forwarded to other system users. The target user will see the chat in their active chat queue after receiving it.
Transfer a chat to another agent

![Transfer a chat to another agent](https://academy.creatio.com/docs/sites/en/files/images/Release_notes/release_notes_7_18_2/scr_transfer_chat.png)

- The communication panel now displays the customer chat history in the channel. This lets agents review the previous conversations without leaving the active chat.

- Creatio will notify the user if their browser blocks site notifications, ensuring the agents will not miss new chat messages.
The notification on the communication panel

![The notification on the communication panel](https://academy.creatio.com/docs/sites/en/files/images/Release_notes/release_notes_7_18_2/scr_browser_notification.png)


### Integrations [​](https://academy.creatio.com/docs/8.x/resources/release-notes/7182-release-notes\#title-2070-8 "Direct link to Integrations")

#### Exchange Listener [​](https://academy.creatio.com/docs/8.x/resources/release-notes/7182-release-notes\#title-2070-9 "Direct link to Exchange Listener")

- Creatio .NET Core now supports Exchange calendar integration.
- The Exchange Listener microservice can now be configured to establish a password-protected connection to the Redis caching server. This helps to standardize the Creatio component deployment.

### Performance [​](https://academy.creatio.com/docs/8.x/resources/release-notes/7182-release-notes\#title-2070-10 "Direct link to Performance")

#### Acceleration [​](https://academy.creatio.com/docs/8.x/resources/release-notes/7182-release-notes\#title-2070-11 "Direct link to Acceleration")

- Package creation is now 10 times faster.
- Creatio performance when 100 or more concurrent users log in via SSO and Just-in-time User Provisioning was improved.

#### Cold audience warmup schedule [​](https://academy.creatio.com/docs/8.x/resources/release-notes/7182-release-notes\#title-2070-14 "Direct link to Cold audience warmup schedule")

- It is now possible to change the email schedule for the "Warmup cold audience" mode. To do this, contact Creatio support. View the schedule in the **Warmup mode** tooltip on the email page after the change.

- Personal responses sent when an email is reopened no longer overwrite the data about the recipient's other actions.
