Release date: 10/06/2025

This document details the technical changes and enhancements introduced in Creatio Financial Services CRM 1.1. It is intended for developers, system administrators, and DevOps engineers responsible for maintaining and extending Creatio customizations.

For a comprehensive overview of the new features, refer to the [1.1 Creatio Financial Services CRM release notes](https://academy.creatio.com/docs/8.x/resources/finserv-release-notes/1-1-finserv-release-notes).

The update guide for the on-site applications is available in a separate [article](https://academy.creatio.com/documents?id=2495).

This change improves governance, simplifies maintenance, and ensures consistent setup of underwriting, product, and reference data across financial service workflows.<br>More details<br>Added the following sections to the "Lending configuration" workplace:<br>- **Underwriting rules** for managing manual and conditional underwriting validation logic.<br>- **Products** for maintaining the product catalog and related parameters.<br>- **Lookups** for configuring reference data used across lending processes. |
