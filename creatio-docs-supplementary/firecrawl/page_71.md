Release date: 10/30/2024

At Creatio, we are committed to empowering our customers with industry-leading product innovations for workflow automation, no-code development, and modern CRM. Today we are advancing new forms of innovation with Creatio version 8.2 "Energy," featuring the **following new capabilities and enhancements**.

The update guide for the on-site applications is available in a separate [article](https://academy.creatio.com/documents?id=2495).

## Creatio AI overview [​](https://academy.creatio.com/docs/8.x/resources/release-notes/82-energy-release-notes\#title-2782-114 "Direct link to Creatio AI overview")

Creatio AI delivers an AI-native architecture that has a robust set of the latest capabilities, all built using simple and intuitive no-code tools. Creatio AI leverages Large Language Models to empower users to complete daily tasks faster and more efficiently.

Creatio AI provides a unified AI environment that combines predictive, generative, and agentic AI capabilities into a single, easy-to-adopt platform for the entire organization. Creatio AI sits at the core of the Creatio platform, seamlessly accessing all essential components and data.

Creatio AI is available to all platform users regardless of subscription level or product.

### Creatio AI: enabling greater end user productivity and efficiency [​](https://academy.creatio.com/docs/8.x/resources/release-notes/82-energy-release-notes\#title-2782-120 "Direct link to Creatio AI: enabling greater end user productivity and efficiency")

Creatio AI intelligently acts as a co-creator together with your users, by providing them with intelligent search of data, responding to user requests based on information available within Creatio and externally across the web, guiding users through complex tasks effortlessly. More importantly, it also supports custom scenarios - known as AI Skills - that allow users to complete specific tasks or automate workflow by following step-by-step guidance crafted by the AI Skill’s creator, ensuring tailored experience. AI Skills thus represent the way Creatio AI learns and becomes smarter. AI Skills can be developed using natural language processing to create and deploy new AI Skills to end-users without writing any code.

### Creatio AI: AI Command Center [​](https://academy.creatio.com/docs/8.x/resources/release-notes/82-energy-release-notes\#title-2782-121 "Direct link to Creatio AI: AI Command Center")

Creatio AI also introduces a new **AI Command Center** workspace that lets you fully customize the AI tools. It consists of the following sections:

- **Creatio AI setup** that lets you configure out-of-the-box AI Skills and add your own skills.
- **Creatio AI dashboards** that provides comprehensive information on which AI Skills are being used, how many tokens have been consumed, who the most active users are, and much more.
- **Predictive (machine learning) models** where you can configure your own machine learning model to meet your specific needs.

AI Skills

![AI Skills](https://d3a7ykdi65m4cy.cloudfront.net/ac-en/s3fs-public/images/Release_notes/release_notes_8_2/scr_ai_skills.png)

Creatio AI’s AI Command Center helps to eliminate the friction of adopting AI across business processes, ensuring you can confidently understand and manage costs. It provides administrators with comprehensive tools for visibility on usage, access management, and scalability to help use Creatio AI with confidence.

### Creatio AI: Pre-built AI Skills [​](https://academy.creatio.com/docs/8.x/resources/release-notes/82-energy-release-notes\#title-2782-122 "Direct link to Creatio AI: Pre-built AI Skills")

Creatio's modern CRM system includes pre-built AI Skills. From predicting customer churn and generating content for knowledge bases to autonomously driving lead qualification processes, it helps businesses reduce operational inefficiencies and achieve powerful productivity gains. More AI Skills will continue to be released, both from Creatio and from our partner ecosystem.

Following are some of the new pre-built CRM skills:

#### Creatio AI Skills for Services [​](https://academy.creatio.com/docs/8.x/resources/release-notes/82-energy-release-notes\#title-2782-123 "Direct link to Creatio AI Skills for Services")

**Case resolution recommendations**. You can now view the recommended next steps to resolve the case using the "Suggest case resolution" Creatio AI Skill. Use it to maximize the chances of a successful case resolution. The AI Skill is available only in the **Case Management** app. The **Knowledge Management** app is required as well.

**Case performance analysis**. You can now view a concise performance analysis report for a case using the new "Case performance" Creatio AI Skill. The report provides key case metrics, escalation reasons, customer satisfaction scores, and a high-level overview of the most significant actions taken by agents to resolve the case. The AI Skill is available only in the **Case Management** app.

**Case summary**. You can now generate a case summary using the new "Summarize case" Creatio AI Skill. The summary provides a brief overview of the current case status, SLA, and recent interactions, highlights escalation reasons, and suggests possible next steps. The AI Skill is available only in the **Case Management** app.

**Articles created from cases**. You can now generate a knowledge base article based on a support case using the new "Create article" Creatio AI Skill. The AI Skill requires both **Knowledge Management** and **Case Management** apps to be installed.

**AI Skill visibility**. You can now view buttons for Creatio AI Skills in Case Management app only on the case form page, case list page, and Service homepage.

#### Creatio AI Skills for Sales [​](https://academy.creatio.com/docs/8.x/resources/release-notes/82-energy-release-notes\#title-2782-124 "Direct link to Creatio AI Skills for Sales")

**Conversion score insights**. You can now learn more about how contact and account conversion scores are calculated using the new "Provide contact conversion insights" and "Provide account conversion insights" Creatio AI Skills. Use these insights to refine targeting, optimize budgets, and enhance collaboration with sales teams for higher conversion rates.

**Lead and opportunity summaries**. You can now obtain a summary of a lead or opportunity using the new "Lead summarization" Creatio AI Skill. The AI Skill includes an overview of the status, customer needs, products of interest, and recent interactions. Also, Creatio AI summarizes the latest customer engagements and feedback, and highlights key information from the feed, actions, emails, and calls. The AI Skill is available only in the **Lead and Opportunity Management** app.

Lead summarization AI Skill

#### Creatio AI Skills for Marketing [​](https://academy.creatio.com/docs/8.x/resources/release-notes/82-energy-release-notes\#title-2782-125 "Direct link to Creatio AI Skills for Marketing")

**Bounce responses**. You can analyze bounces received from an individual recipient within a bulk email campaign by using the new "Analyze bounce responses" Creatio AI Skill. The analysis includes a detailed explanation of why the bounce occurred and recommendations on how to address it. The AI Skill is only available in the **Email Marketing** app.

**Enhanced email subject in bulk emails**. You can optimize your email subject lines using the "Enhance bulk email subject lines" Creatio AI Skill. The Skill identifies common spam words, for example, "Urgent" or "Free" and phrases that create urgency, for example, "Verify your account," and suggests safe alternatives to generate high-engagement subject lines. This ensures your emails are more likely to land in inboxes rather than spam folders. The AI Skill is available only in the **Email Marketing** app.

Enhance bulk email subject lines AI Skill

![Enhance bulk email subject lines AI Skill](https://d3a7ykdi65m4cy.cloudfront.net/ac-en/s3fs-public/images/Release_notes/release_notes_8_2/scr_copilot_email_subject.png)

**Text rewriting**. You can now rewrite text using Creatio AI in any rich text field and other components that support rich text, including **Feed** and **Message composer** components. Simply select the text and pick an available option. Whether you need to simplify complex ideas, enhance readability, or adjust tone, Creatio AI rewrites your text while preserving its original meaning. This is perfect for content creators, marketing teams, and professionals looking to optimize communication. Creatio AI ensures your message is clear, concise, and impactful.

**Text translation**. You can translate text to over 100 languages quickly and accurately using Creatio AI. Advanced AI preserves tone and meaning, making it ideal for businesses, global teams, and customer interactions.

#### Creatio AI Skills for Productivity [​](https://academy.creatio.com/docs/8.x/resources/release-notes/82-energy-release-notes\#title-2782-126 "Direct link to Creatio AI Skills for Productivity")

**Meeting scheduling**. You can effortlessly schedule a new meeting by using the new "Plan meeting" Creatio AI Skill. The Skill automatically identifies available time slots and schedules meetings for all required attendees. The AI Skill is available only in the **Productivity** app.

#### Creatio AI Skills for Studio [​](https://academy.creatio.com/docs/8.x/resources/release-notes/82-energy-release-notes\#title-2782-127 "Direct link to Creatio AI Skills for Studio")

**Send email**. You can now send an email from Creatio quickly and easily by using the new "Send email" Creatio AI Skill.

**Learn more about Creatio AI**. You can now learn more about Creatio AI functionality from Creatio AI itself by using the new "What you can" Creatio AI Skill.

## End user experience​ [​](https://academy.creatio.com/docs/8.x/resources/release-notes/82-energy-release-notes\#title-2782-8 "Direct link to End user experience​")

**Virtual horizontal scroll**. Creatio now loads only the visible columns in the **List** component and loads other columns as you scroll horizontally.

**UX improvements for navigation panel**.

- Creatio now explains why the navigation menu is empty, for example, due to lack of permissions.
- Creatio now states that a section search within the workplace yielded no results clearly.
- Search functionality in the navigation panel now fetches sections from all workplaces available to you, even if the sections are not visible in the current workplace.

**Remember me**. You can now set up the username autofill functionality on the level of an individual user.

**Accessibility compliance improvements**.

- Creatio now specifies the default human language in Freedom UI.
- Freedom UI pages are now separated into sections using landmark roles.

**New button style**. You can now use the **Outline** button style.

### Integrations [​](https://academy.creatio.com/docs/8.x/resources/release-notes/82-energy-release-notes\#title-2782-12 "Direct link to Integrations")

**OAuth integrations**. You can now manage OAuth integrations more easily using the new Freedom UI OAuth section in the System Designer. When you create an integration, Creatio automatically adds and assigns it a dedicated technical user to ensure security.

When you connect Creatio to Identity Service, you no longer need to create a default resource manually.

Learn more: [Manage the OAuth 2.0 client credentials](https://academy.creatio.com/documents?id=2508).

**New API for message channel service**. You can now use web sockets or SignalR with Freedom UI pages using a new messageChannelService API. The API abstracts the underlying implementation, which enables seamless communication without dependencies on the specific underlying technology.

**Improvements to technical users**.

- The functionality is now available publicly.
- Restrict IP addresses that a technical user can use to interact with Creatio.
- End one or more sessions of a technical user on the form page of the user.
- Manage roles of the technical user on the form page of the user.

## Advanced customization [​](https://academy.creatio.com/docs/8.x/resources/release-notes/82-energy-release-notes\#title-2782-13 "Direct link to Advanced customization")

**Storage for user consent history**. You can now access user consents that can be used in any custom functionality from "Consent" and "User consent history" objects. Administrators can extract data from the objects to track and manage user agreements. Creatio AI UI and Marketing Designer features also use this storage. Learn more: [User consent storage](https://academy.creatio.com/documents?id=15165).

**Creatio AI improvements**.

- Use AI Skills in the front-end via a new typescript API.
- Use AI Skills in client-server web architecture via a new HTTP API.
- Customize Creatio AI more extensively using input parameters for the API AI Skills.

Learn more: [Handle system-level data using Creatio AI](https://academy.creatio.com/docs/8.x/dev/development-on-creatio-platform/category/creatio-ai).

## Administration​ [​](https://academy.creatio.com/docs/8.x/resources/release-notes/82-energy-release-notes\#title-2782-14 "Direct link to Administration​")

**Recovery codes**. You can now generate recovery codes for two-factor authentication. Use the codes in case you lose access to other verification methods.

**Permissions for administrated operations**. You can now grant access to objects based on permissions for a system operation regardless of whether the user has access on the object level. Perform the setup either in the **Object permissions by operations** lookup or on the **Operation permissions** page.

**Permission warning**. When you deactivate operation permissions for an object, Creatio now warns the administrator about access rules that will be applied for company employees, external users, and technical users.

**First day of the week**. It is now possible to specify the first day of the week to display in Creatio independently from a chosen language. Users can specify it in the user profile, administrators can do it in the "Default first day of the week" ("DefaultFirstDayOfTheWeek" code) system setting.

**Automatic customization deployment in the web farm**. Creatio now deploys customization to all web farm nodes and synchronizes them automatically. This ensures consistent updates and minimizes the risk of errors. Customizations that do not require nodes to be restarted can now be applied during business hours without taking Creatio offline or scheduling maintenance windows. Before you can take advantage of this functionality, apply a new approach to web farm deployment.

**Improved page loading time**. Creatio now loads data of expandable lists only if they appear in the viewport.
