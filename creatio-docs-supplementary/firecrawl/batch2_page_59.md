Version: 8.0All Creatio products

On this page

Software products on the Creatio platform are web applications, so they do not require any installation on the client side. Users access the application via an Internet browser. The minimum monitor resolution required to access Creatio is 1280x768.

note

Learn more about the recommended parameters for the mobile apps: [System requirements for mobile devices](https://academy.creatio.com/documents?product=mobile&ver=7&id=1945).

The recommended data channel for the client-server connection is 3 Mbps per one active user.

One of the following Internet browsers must be installed on the client-side:

|     |     |
| --- | --- |
| Microsoft Windows:<br>- Windows 10<br>- Windows 11 | Google Chrome, the latest official version on the Creatio release date |
| Mozilla Firefox, the latest official version on the Creatio release date |
| Microsoft Edge, the latest official version on the Creatio release date |
| OS X, the latest official version. | Apple Safari, the last official version on the Creatio release date |

note

Release dates are available in the [release calendar](https://academy.creatio.com/node/1820/).

Important

Using third-party plug-ins and add-ons for browsers might increase the page loading time and cause pages to display incorrectly.
